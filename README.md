# Lumex Movie Search - Поиск и воспроизведение фильмов

Веб-приложение для поиска и воспроизведения фильмов и сериалов с использованием Lumex API.

## Возможности

- 🔍 Поиск фильмов и сериалов по названию
- 🎬 Воспроизведение видео в высоком качестве (до 1080p)
- 🌐 Поддержка множественных переводов
- 📱 Адаптивный дизайн для всех устройств
- 🎯 Интеграция с Lumex API
- 📺 HLS видеоплеер с поддержкой потокового воспроизведения

## Технологии

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **UI Framework**: Bootstrap 5
- **Видеоплеер**: HLS.js для поддержки HLS потоков
- **API**: Lumex Portal API + Lumex Stream API
- **Авторизация**: OAuth2

## Установка и запуск

1. Скачайте все файлы проекта
2. Откройте `index.html` в веб-браузере
3. Приложение готово к использованию!

## Использование

### Поиск фильмов
1. Введите название фильма или сериала в поле поиска
2. Нажмите кнопку "Поиск"
3. Выберите фильм из результатов поиска

### Воспроизведение
1. Кликните на карточку фильма
2. Выберите желаемый перевод из выпадающего списка
3. Видео начнет воспроизводиться автоматически
4. Используйте стандартные элементы управления плеера

### Управление плеером
- **Закрытие плеера**: Кнопка "X" или клавиша Escape
- **Смена перевода**: Выпадающий список в правом верхнем углу
- **Полноэкранный режим**: Кнопка в плеере

## API Интеграция

Приложение использует два API:

### 1. Lumex Portal API (поиск)
- **Endpoint**: `https://portal.lumex.host/api/short`
- **Метод**: GET
- **Параметры**: `api_token`, `title`, `page`, `limit`

### 2. Lumex Stream API (воспроизведение)
- **Авторизация**: `https://api.lumex.pw/login`
- **Обновление токена**: `https://api.lumex.pw/refresh`
- **Получение потока**: `https://api.lumex.pw/stream`
- **Получение ссылки**: `https://api.lumex.pw/verify/{playlist}`
